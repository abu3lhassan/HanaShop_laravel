# HanaShop Laravel

HanaShop is an elegant e-commerce project for decor, kitchen tools, and electronics.

## Project sections

- Premium public store interface
- Product categories
- Product details
- Admin dashboard
- Products management
- Customer management
- Invoice overview

## GitHub Pages

The repository includes a ready `index.html` page in the project root. Upload the project to GitHub and enable GitHub Pages from the repository root to show the public HanaShop page.

## Laravel application

The Laravel application files are also included for local PHP hosting or a Laravel-compatible server.
